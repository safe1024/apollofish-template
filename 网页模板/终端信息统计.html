查看详细步骤

Windows系统：

  • 方法1：按 Win + R 键，输入 cmd 回车，在命令窗口输入 ipconfig，查找"IPv4 地址"
  • 方法2：打开设置 → 网络和Internet → 查看网络属性
  • 方法3：在搜索框输入"网络连接" → 双击当前连接 → 点击"详细信息"

macOS系统：

  • 方法1：点击左上角苹果菜单 → 系统偏好设置 → 网络 → 选择当前连接
  • 方法2：打开终端，输入命令 ifconfig | grep "inet "

Linux系统：

  • 打开终端，输入命令 ip addr show 或 ifconfig

通用方法：

  • 访问网站：whatismyipaddress.com
  • 在浏览器搜索"我的IP地址"
